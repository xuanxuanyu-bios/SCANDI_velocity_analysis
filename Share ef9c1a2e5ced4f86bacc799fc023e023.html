Step 2, run velocyto on hipergator:

By running the following slurm, we can generate .loom file for each of the samples we had. 

# Path to the directory containing sample subfolders
samples_dir=/orange/paefron/dayuan.wang/mice_cite-seq_preprocessing/cellranger/neonatal/output_20231114

# Load conda module and activate environment
module load conda
conda activate tutorial

module load samtools/1.9	

# Path to the GTF file
gtf_path=/blue/paefron/dayuan.wang/NGS_ICBR_mice/Preprocessing/Cell_Ranger/reference/refdata-gex-mm10-2020-A/genes/genes.gtf

# Loop over each sample folder and run velocyto
for sample_dir in "$samples_dir"/*; do
    if [ -d "$sample_dir" ]; then
        echo "Running velocyto on $(basename "$sample_dir")"
        velocyto run10x "$sample_dir" "$gtf_path"
    fi
done
